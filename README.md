# NexTwinAI

Digital twin and AI-driven manufacturing analytics platform scaffold.

## Structure

- `frontend/` - UI app and components
- `backend/` - FastAPI backend services and database
- `ai-engine/` - ML model training, inference, and evaluation
- `digital-twin/` - simulation, state, and visualization assets
- `edge-device/` - IoT sensor and edge deployment tools
- `docs/` - project documentation
- `tests/` - unit and integration tests
